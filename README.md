🎀 DoroPet: AI 智慧桌面小管家DoroPet 是一個基於 Python 與 PyQt6 開發的智慧互動桌面寵物。它具備「地端優先」的混合 AI 大腦，能陪你聊天、查詢美股資訊，還會記錄你們之間的所有點滴。✨ 核心功能混合 AI 動態分流：優先使用地端 Ollama (Gemma2:2b) 進行快速回覆，遇到複雜問題（如股市分析）時自動切換至 Gemini 2.5 Flash。自動化服務管理：啟動程式時會自動檢查並背景喚醒 Ollama 服務，若缺少模型也會自動執行安裝。互動式體驗：自由拖曳：隨意將 Doro 置於桌面的任何角落。雙擊對話：快速開啟對話框，支援中文輸入，並具備防手抖重複發送機制。回憶系統：右鍵選單可查看完整的歷史紀錄視窗，文字清晰且自動分類。動態跟隨氣泡：對話氣泡會自動黏著在 Doro 頭頂，並隨移動即時更新位置。🚀 快速啟用步驟1. 安裝 Ollama (地端大腦核心)前往 Ollama 官方網站 下載並安裝。安裝完成後，建議先開啟終端機執行一次 ollama run gemma2:2b 確保模型可正常運作（之後 Doro 會自動管理）。2. 環境設定確保你的電腦已安裝 Python 3.10+。複製專案到本地端後，安裝必要套件：Bashpip install PyQt6 google-genai python-dotenv requests yfinance
在專案根目錄建立 .env 檔案，並填入你的 API 金鑰：程式碼片段GEMINI_API_KEY=你的_GEMINI_API_KEY
執行程式直接執行主程式即可：Bashpython main.py
🎮 使用說明動作描述左鍵拖曳抓起 Doro 進行移動。左鍵雙擊彈出對話輸入框，支援中文輸入法（按 Enter 發送，Esc 關閉）。右鍵點擊開啟功能選單，可選擇「查看回憶」或「關閉程式」。點擊旁邊輸入框與歷史視窗在失去焦點時會自動優化顯示狀態。📂 專案架構core/：包含視窗邏輯、事件匯流排與服務管理。features/：包含大腦分流 (Brain)、聊天 UI 與歷史紀錄儲存等模組。assets/：存放 Doro 的左右鏡像動畫素材。🛠️ 開發備註本專案特別針對 Windows IME（中文輸入法）進行了事件優化，防止重複發送請求。使用了 subprocess.CREATE_NO_WINDOW 旗標，確保地端 AI 服務在背景安靜執行而不跳出黑視窗。
